orange-standard

A headless Bitcoin Mempool MCP server for AI agents. It exposes real-time Bitcoin network state from mempool.space over the Model Context Protocol — no browser, no UI.

Endpoint

Connect an MCP client (Streamable HTTP) to:

POST /mcp

Tools

🧱 get_latest_blocks — most recent blocks (height, hash, miner, tx count)

🔎 get_block_detail — full metadata for a block by height or hash

📊 get_mempool_status — pending tx count, size, projected blocks

💸 get_fee_estimates — recommended fee rates (sat/vB)

See the repository for setup.
